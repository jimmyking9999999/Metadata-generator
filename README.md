This repo serves as an automatic updater for `nexusmods.json` 

`nexusmods.json` is refreshed automatically from the Nexus Mods API by `.github/workflows/update-nexusmods.yml`.
Please see the autogenerated file for all fields obtained. This script is ran once/day by default.
